# Simple Calculator

A simple command-line calculator that performs basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Requirements
- Python 3.x

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/simple-calculator.git
   ```

2. Navigate to the project directory:
   ```bash
   cd simple-calculator
   ```

## Usage

1. Run the `calculator.py` file:
   ```bash
   python calculator.py
   ```

2. Follow the prompts to enter two numbers and select an operation.

### Example:
```bash
Enter first number: 5
Enter second number: 3
Select operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division
Enter your choice: 1
Result: 8
```
